This is a collection of Vue 3.0 composition API functions and utilties to allow your apps to adapt your user's:

- Network conditions.
- Device Memory.
- CPU Cores.

## Install

```sh
yarn add vue-adaptive-utils

# or

npm i vue-adaptive-utils --save
```
